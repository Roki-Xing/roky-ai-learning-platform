# Sprint 17 Reflection

## 结果

Sprint 17 已完成本地实现和生产上线：`/voice` 的音频文件可以提交到服务端，服务端会校验大小和 MIME 类型，并在配置 `OPENAI_API_KEY` 时调用转写接口；无 key、测试环境或转写失败时显式回退到手动 transcript。

## 保持边界

- 不做实时语音。
- 不长期保存音频文件。
- 不新增数据库字段。
- 不暴露 API key。

## 后续

- 生产当前未配置 `OPENAI_API_KEY`，自动转写功能处于可用但未启用状态。
- 后续可继续做语音转写状态展示、转写错误 UI 和真正的语音到 Coach 一键流程优化。
